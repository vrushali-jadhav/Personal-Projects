Projects (A list of academic projects):

  1. Patient data viewer (Group project):

      An mobile application that let's an authenticated doctor log into the app.
      Check patients information:

    • Medical history
    • Medical reports such as xrays, sonography videos.
    • Write new prescription.
    • Call patient or patients family members.

      • Technologies: Java, SQL, Andoroid.


  2. Scambio (Group project):

      A website that lets you exchange items. User can:

    • Explore the website without registration.
    • Signup and login.
    • Upload items for exchange.
    • Choose an another users item for exchange in return for any of his item.
    • It's a match if the other user has also liked/selected your item for exchange.
    • Post a 'match', users can chat and decide on location or way to exhange the items.

    Technologies: Mongo, Mongoose, Express, Node.js


  3. Starbucks - Cloud Technologies (Group project):

      An order management system. Description:

    • APIs to place order, pay, checkout, add and explore reviews written in GO.
    • 6 MongoDB clusters hosted in AWS used as backend.
    • GO APIs deployed in dockers in AWS EC2 instances (Private network.)
    • AWS load balancer connecting all the dockers to the outside world.
    • Kong API gateway connected to load balancer.
    • Front end (node.js) deployed in a docker machine in Public network, connecting to backend via Kong API gateway.

    Technologies: Mongo, AWS, Docker, Kong API, GO, Node.js


  4. Pig game (Hobby project):

      Description:

    • 2 player game.
    • Players can enter the winning score.
    • Roll the dice, get the score.
    • If you roll a 1, you lose the points.
    • 1st player to reach the winning score, wins the game!

    Technologies: JavaScript, CSS3, HTML5
